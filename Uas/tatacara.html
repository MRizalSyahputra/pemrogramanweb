Aturan penulisan nomor seri buku

  1. 2 digit pertama menunjukkan tanggal input
    (jika tanggal input hanya 1 digit, isikan 0 di depannya. Misal tanggal 5, tuliskan 05)
  2. 2 digit selanjutnya menunjukkan jenis buku :
    • FX = fiksi
    • NF = nonfiksi
    • BG = biografi
  3. 2 digit selanjutnya menunjukkan genre paling menonjol:
    • untuk buku nonfiksi atau biografi
      • 74 = self-improvement/motivasi
      • 39 = gaya hidup
      • 42 = tutorial
      • 15 = teknologi
      • 82 = sosial
      • 35 = ekonomi
      • 94 = seni
      • 26 = almanak/ensiklopedia
      • 18 = sejarah
      • 91 = buku panduan
      • 40 = psikologi
      • 07 = olahraga
      • 03 = kesehatan
      • 50 = agama
      • 33 = musik
      • 99 = lain-lain
    • fiksi :
      • D7 = aksi
      • F3 = petualangan
      • J2 = fantasi
      • R9 = romansa
      • S1 = sci-fi
      • B8 = thriller/horror
      • P0 = paranormal
      • C6 = keseharian/slice of life
      • H4 = misteri
      • D5 = dewasa
      • M4 = militer
      • L2 = sihir
      • G9 = drama
      • K9 = komedi
      • ZZ = lain-lain
    • tidak tahu atau tidak disebutkan = 01
  4. 2 digit selanjutnya menunjukkan lokasi terbit & bahasa yang digunakan:
    • 45 = Indonesia
    • Luar indonesia :
      • 3N = Bahasa Inggris
      • D3 = Bahasa Jerman
      • J6 = Bahasa Jepang
      • 6R = Bahasa Perancis
      • KR = Bahasa Korea
      • P8 = Bahasa Portugis
      • I0 = Bahasa Italia
      • AR = Bahasa Arab
      • GL = Bahasa Lainnya : GL
    • 98 = buku luar negeri terjemahan
  5. digit sisanya merupakan identitas random yang bisa anda karang untuk membedakan dengan buku lain jika ada yang formatnya sama
